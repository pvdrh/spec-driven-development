# Auth Service — Hướng dẫn Phát triển

Dự án sử dụng **Spec-Driven Development (SDD)**: Spec dẫn dắt Code.

## 🎯 Quy trình

Khi phát triển feature mới:

1. **Tạo 4 file spec** trong `specs/features/`:
   - `BR-XXX-<tên>.md` — Business Requirement
   - `UC-XXX-<tên>.md` — Use Cases
   - `EM-XXX-<domain>.md` — Entity Model
   - `AC-XXX-<tên>.md` — Acceptance Criteria

2. **Lên plan & code** theo Spec (không vá tay)

3. **Test naming:** `test_UC_<id>_AC_<number>_<scenario>`
